# Mi Portfolio

Este es mi portfolio personal para promocionar mis proyectos a clientes y empleadores.

## Cómo ejecutar

1. Instala las dependencias: `npm install`
2. Ejecuta el servidor de desarrollo: `npm run dev`
   - Si PowerShell no reconoce `npm`, ejecuta: `./run-dev.ps1`
3. Abre [http://localhost:3000](http://localhost:3000) (o el puerto que indique la consola, por ejemplo `3001`) en tu navegador.

## Tecnologías

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Demo de E-Commerce

La demo de tienda se encuentra en `/store`, con productos, páginas de detalle y un carrito básico.

## Task Management App

La app de gestión de tareas está en `/projects/task-management-app`, con funcionalidades para crear, editar, eliminar tareas, categorías y prioridades.

## Data Visualization Dashboard

El dashboard de visualización de datos está en `/projects/data-visualization-dashboard`, con gráficos interactivos (barras, líneas, pastel), métricas en tiempo real y exportación de datos.

## Landing Page Pro

La landing page moderna está en `/projects/landing-page-pro`, con secciones de hero, features, testimonials, formulario de contacto y diseño responsive.
